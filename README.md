# OAUSTECH Admission Eligibility Portal (Technical Documentation)

### 📌 Project Overview
This portal is a technical demonstration developed to assist the **Olusegun Agagu University of Science and Technology (OAUSTECH)** Admissions Unit in automating initial aspirant screening. It provides real-time eligibility feedback based on official UTME scoring thresholds.

### 🛠️ Technical Specifications
- **Architecture:** Client-side Logic Engine (Vanilla JavaScript)
- **UI Framework:** Tailwind CSS (Institutional Grade)
- **Deployment:** GitHub CI/CD Pipeline
- **Optimization:** Mobile-First Responsive Design

### 🎯 Objective
To reduce the administrative load on the ICT and Admissions departments by providing aspirants with a self-service tool for course eligibility checking.
